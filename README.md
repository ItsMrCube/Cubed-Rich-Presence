### Setup Guide

1. Install [NodeJS](https://nodejs.org).
1. Run `npm i`.
1. Create a `config.js` file.
1. Fill it with this template:

```js
module.exports = {
  // Create an app and get the ID at https://discord.com/developers/applications

};
```
